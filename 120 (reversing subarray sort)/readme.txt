120 - Stacks of Flapjacks
https://onlinejudge.org/index.php?option=com_onlinejudge&Itemid=8&page=show_problem&problem=56

Each array represents a stack of pancakes.
The sorting task is to make an ascending array by reversing ("flipping") left sub-array.
The flip indices are counted from right to left.

The output shows the input array and the sequence of the flips to make it sorted.
