299 - Train Swapping
https://onlinejudge.org/index.php?option=com_onlinejudge&Itemid=8&category=98&page=show_problem&problem=235

The train carriages are sorted using rotating railway bridge with the length of 2 carriages.
The task is to count the number of swappings during sorting.

According to the procedure in the statement, this is bubble sort.
